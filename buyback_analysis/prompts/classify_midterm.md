あなたは、企業のIR文書を分類するアシスタントです。
以下の文書は中期経営計画に関するIRですが、定量目標（metrics）が抽出できませんでした。
文書の内容を読んで、その理由を以下の3値で分類してください。

### 分類基準

| 値 | 説明 |
|---|---|
| `withdrawn` | 中期経営計画の取り下げ・廃止通知（既存データが無効になるもの） |
| `no_targets` | 定量目標が記載されていないもの |
| `postponed` | 中期経営計画の公表延期お知らせ（計画そのものの開示を先送りする通知） |

### 重要な判断ルール

**`no_targets` にすべきケース（迷ったらこちら）：**
- 中期経営計画の「策定に関するお知らせ」「策定および〇〇について」など、計画を発表した文書だが数値目標がないもの
- 「（訂正）〇〇について」などの訂正通知で数値なし
- 別PDFへのリンクのみで数値が記載されていないもの

**`postponed` にすべきケース：**
- 「延期」「見送り」などの表現があり、計画の公表そのものを先送りにする通知
- 計画の内容（数値目標・計画名・期間）が一切ない純粋な延期お知らせ

**`withdrawn` にすべきケース：**
- 計画の取り下げ・廃止を明示している通知

### 出力形式

```json
{{
    "extraction_status": "withdrawn"
}}
```

`extraction_status` には `"withdrawn"` / `"no_targets"` / `"postponed"` のいずれかのみを入れてください。
