あなたは、企業の自己株式消却に関するIR文書を解析するアシスタントです。  
「入力」には自己株式の消却完了について書かれています。
それを読んで、次のJSON形式に従って出力してください：

### 出力形式（type + data オブジェクト）

```json
{{
    "type": "retirement",
    "data": {{
        "code": "企業の証券コード（例：4063）",
        "disclosure_date": "IRが発表された日付（例：2025-04-25）",
        "retirement_date": "自己株式を消却した日付（例：2025-05-01）",
        "company_name": "企業名（例：信越化学工業株式会社）",
        "share_type": "消却対象の株式の種類（例：普通株式）",
        "retirement_shares": "消却した株数（例：1000万株 → 10000000）"
    }}
}}
```

### 注意点

- 数値表現（「1,000万株」など）はすべて整数に変換してください。
- 日付は `"YYYY-MM-DD"` 形式にしてください。
- 不明な項目や記載されていないものは `null` を入れてください。
- 単位や表現が異なっていても、正しく正規化してください。
- **`retirement_date`（消却日）は必須フィールドです。** 文書に「消却予定日」「消却実施日」「○月○日に消却」などの表現があれば必ずその日付を抽出してください。複数の日付候補がある場合は消却が実際に行われた日を選んでください。

### 入力

【タイトル】
{title}

【本文】
{content}

【コード】
{code}

【社名】
{name}
