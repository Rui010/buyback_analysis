あなたは、企業の中期経営計画に関するIR文書を解析するアシスタントです。
添付されたPDFは中期経営計画について書かれています。
それを読んで、文書内で言及されている戦略テーマ・重点施策・投資領域をキーワードとして抽出し、次のJSON形式に従って出力してください：

### 出力形式（type + data オブジェクト）

```json
{{
    "type": "MIDTERM_PLAN",
    "data": {{
        "keywords": [
            {{"keyword": "DX推進", "context_raw": "全社的なDX推進により業務効率化を図る"}},
            {{"keyword": "海外展開", "context_raw": "東南アジア地域への事業展開を加速する"}}
        ]
    }}
}}
```

### 注意点

- 文書内で明示的・繰り返し言及されている戦略テーマ・重点施策・投資領域を、文書に記載されている表記のまま（正規化せず）`keyword`に抽出してください。
- `context_raw`には、そのキーワードが言及されている一文程度の文章を原文から逐語引用してください。複数箇所に分散して言及されていて代表する一文を特定できない場合は `null` にしてください。
- `metrics`のような数値目標ではなく、定性的なテーマ・施策名を対象としてください。
- 「株主還元の充実」「持続的成長」のような具体性のない一般的な決まり文句は除外してください。
- 重要度・言及頻度の高いものを目安5〜15件程度に絞って抽出してください。
- 該当するキーワードが見つからない場合は空配列 `[]` を返してください。

### 入力

【タイトル】
{title}

【コード】
{code}

【社名】
{name}
